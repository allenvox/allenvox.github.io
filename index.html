My long and winding road

I got into programming when I was about 12.

Like the most of us, I started with simple Python games and HTML pages. I was attracted to different technologies — from desktop forms on Devel, Java and plugins for the Bukkit server core, web applications and bots on NodeJS and Go, to mobile iOS applications on Swift and gamedev with Unity and C#.

At my alma mater, I embarked on the path of a system architect and developer, studying mathematics in depth (including differential equations and functional analysis), computer science, distributed computing systems, parallel and high-performance computing using C and C++. I was also interested in Machine Learning and neural networks, which I defined for my final diploma topic, making models for detecting fraudulent transactions in online banking with Python.

I found the opportunity in October 2024 on a career day at the city scientific library, getting a job as an intern C++ developer at Eltex. And I'm still here! Developing myself into a better version of myself, I hope.
